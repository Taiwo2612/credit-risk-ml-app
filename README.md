# Credit Risk Prediction App

This is a **Credit Risk Prediction ML App** built using Python and Streamlit.  
It predicts whether a loan applicant is a **low-risk (good) or high-risk (bad) borrower** based on personal and financial details.

## Features
- Input applicant details: Age, Sex, Job, Housing, Saving & Checking accounts, Credit Amount, Duration, Purpose
- Predicts Credit Risk using a **Random Forest Classifier**
- Web interface built with **Streamlit**
- Model trained on the **German Credit Dataset**

## Files in this Repository
- `app.py` — Streamlit app   
- `credit_risk_model.pkl` — Trained Random Forest model  
- `model_columns.pkl` — Columns used for model  
- `requirements.txt` — Python dependencies
